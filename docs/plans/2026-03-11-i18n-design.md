# mall4w 前端 i18n 国际化设计文档

## 概述

为 mall4w（Vue 3 电商前端）实现多语言国际化支持，采用 vue-i18n 方案。

## 需求

- **范围**：仅前端 mall4w 项目
- **支持语言**：简体中文 (zh-CN)、英文 (en-US)、日语 (ja-JP)、繁体中文 (zh-TW)
- **切换方式**：自动检测浏览器语言 + 手动切换

## 技术方案

### 核心实现

#### i18n 实例 (locales/index.js)

```js
import { createI18n } from 'vue-i18n'
import zhCN from './zh-CN.json'
import enUS from './en-US.json'
import jaJP from './ja-JP.json'
import zhTW from './zh-TW.json'

const messages = {
  'zh-CN': zhCN,
  'en-US': enUS,
  'ja-JP': jaJP,
  'zh-TW': zhTW
}

// 浏览器语言映射
const localeMap = {
  zh: 'zh-CN',
  'zh-cn': 'zh-CN',
  'zh-tw': 'zh-TW',
  'zh-hk': 'zh-TW',
  en: 'en-US',
  'en-us': 'en-US',
  ja: 'ja-JP',
  'ja-jp': 'ja-JP'
}

function detectBrowserLocale() {
  const lang = navigator.language.toLowerCase()
  return localeMap[lang] || localeMap[lang.split('-')[0]] || null
}

function getStoredLocale() {
  return localStorage.getItem('locale')
}

function setStoredLocale(locale) {
  localStorage.setItem('locale', locale)
}

const i18n = createI18n({
  legacy: false,
  locale: getStoredLocale() || detectBrowserLocale() || 'zh-CN',
  fallbackLocale: 'zh-CN',
  messages
})

export { i18n, setStoredLocale, getStoredLocale, detectBrowserLocale }
export default i18n
```

#### useLocale Composable (composables/useLocale.js)

```js
import { computed } from 'vue'
import { useI18n } from 'vue-i18n'
import { i18n, setStoredLocale } from '@/locales'

const localeOptions = [
  { value: 'zh-CN', label: '简体中文' },
  { value: 'en-US', label: 'English' },
  { value: 'ja-JP', label: '日本語' },
  { value: 'zh-TW', label: '繁體中文' }
]

export function useLocale() {
  const { locale, t } = useI18n()

  const currentLocale = computed(() => locale.value)

  const changeLocale = (newLocale) => {
    locale.value = newLocale
    setStoredLocale(newLocale)
    // 同步更新 Element Plus 语言
    document.documentElement.setAttribute('lang', newLocale)
  }

  return {
    locale,
    currentLocale,
    localeOptions,
    changeLocale,
    t
  }
}
```

#### LocaleSwitch 组件 (components/common/LocaleSwitch.vue)

```vue
<template>
  <el-dropdown @command="changeLocale">
    <span class="locale-switch">
      {{ currentLocaleLabel }}
      <el-icon><ArrowDown /></el-icon>
    </span>
    <template #dropdown>
      <el-dropdown-menu>
        <el-dropdown-item
          v-for="item in localeOptions"
          :key="item.value"
          :command="item.value"
          :class="{ active: locale === item.value }"
        >
          {{ item.label }}
        </el-dropdown-item>
      </el-dropdown-menu>
    </template>
  </el-dropdown>
</template>

<script setup>
import { computed } from 'vue'
import { ArrowDown } from '@element-plus/icons-vue'
import { useLocale } from '@/composables/useLocale'

const { locale, localeOptions, changeLocale } = useLocale()

const currentLocaleLabel = computed(() => {
  return localeOptions.find(item => item.value === locale.value)?.label || '简体中文'
})
</script>
```

### Element Plus 国际化集成

```js

import ja from 'element-plus/dist/locale/ja.mjs'
```

## 注意事项

1. **翻译键命名规范**：`模块.子模块.键名`，如 `auth.login.title`
2. **避免动态拼接**：使用插值而非字符串拼接，如 `t('cart.items', { count })`
3. **复数处理**：vue-i18n 支持复数形式，注意英文等语言的复数变化
4. **图片文案**：如果图片包含文字，需要准备多语言版本
